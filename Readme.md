# Stroke Lesion Segmentation Tool

This application is designed to perform **automated segmentation and detection of stroke lesions in brain MRI images**.

It provides both a **Graphical User Interface (GUI)** and a **Command Line Interface (CLI)** to support flexible workflows:

- **Process data** efficiently with automated model inference  
- **Manage models** used for segmentation  
- **Review and export results** for further analysis  

## Requirements
- A modern operating system (Windows 10 or more recent)  
- Sufficient GPU/CPU and RAM resources (Recommended : 16GB)
