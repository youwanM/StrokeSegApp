# Stroke Lesion Segmentation Tool

The Stroke Lesion Segmentation Tool is an application for **automated segmentation and detection of stroke lesions in brain MRI images**.  

It combines state-of-the-art deep learning methods with a simple interface to make advanced neuroimaging analysis more accessible.  

## Key Features
- **Automated lesion segmentation** from brain MRI (T1-w and FLAIR) scans  
- **Graphical User Interface (GUI)** for ease of use  
- **Command Line Interface (CLI)** for integration into pipelines  
- **Model management tools** for updating segmentation models  
- **Result visualization and export** for research and clinical workflows  

## Target Users
- Medical researchers studying cerebrovascular diseases  
- Clinicians seeking assistance in imaging-based stroke analysis  
- Developers building neuroimaging applications or pipelines
