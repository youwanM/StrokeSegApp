# Welcome to the Stroke Lesion Segmentation Tool Installer

This application helps automate the **segmentation and detection of stroke lesions**  
in brain MRI images.  

It provides both a **user-friendly interface** and a **command-line workflow**  
to support researchers, clinicians, and developers.  

The installer will guide you through the setup process.  

For more information, please refer to the included documentation.
